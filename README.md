# Sales Performance Analysis Dashboard (Excel Project)

## Project Overview

This project analyzes a sales dataset using Microsoft Excel to identify business performance trends. The data was cleaned, structured, and analyzed using Pivot Tables and Excel formulas. An interactive dashboard was created to visualize key metrics like sales, profit, and order distribution.

## Objectives

* Analyze sales performance by category, state, and customer segment
* Identify high performing product categories
* Build an interactive dashboard for business insights
* Practice important Excel formulas used in data analysis

## Tools & Skills Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Data Preparation
* Dashboard Design
* XLOOKUP / VLOOKUP
* SUMIFS / COUNTIFS
* IF / IFS
* Slicers and KPI Metrics

## Key Features

* Interactive Sales Dashboard
* KPI metrics (Total Sales, Total Profit, Total Orders, Total Quantity)
* Category, State, and Segment performance analysis
* Dynamic filtering using slicers
* Important Excel formulas demonstrated in a separate formulas sheet

## Key Insights

* Laptop category generated the highest sales revenue
* Consumer segment contributed the majority of sales
* Certain states showed significantly higher sales performance
* Standard shipping mode was frequently used

## Project Structure

Raw_Data → Original dataset
Analysis → Pivot table analysis
Dashboard → KPI dashboard and charts
Formulas → Important Excel formulas used
